# FB_CLIPOL
Freebasic recorte de polígonos

Rutinas para recortes con polígonos, como unión, diferencia, intersección, etc.

Está basado en estas rutinas -> https://github.com/rickbrew/GeneralPolygonClipper
Debemos compilar para conseguir la DLL necesaria. Yo he usado GCC
